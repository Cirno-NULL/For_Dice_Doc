# 基础

## 注释

```lua
--最基础的注释


--[[-----基础多行注释----]]
--[[
print("多行注释")
print("多行注释")
]]



--[[------单节开关------]]
--[[]]
print("多行注释")
print("多行注释")
--]]

--[[
print("多行注释")
print("多行注释")
--]]

--[[----双刀开关注释-----]]
--[=[版本A]=]
print("多行注释A")
--[[-[版本B]=]
print("多行注释B")
--]-]]


--[=[版本A]-=]
print("多行注释A")
--[[-[版本B]=]
print("多行注释B")
--]-]]
```
