# 基础

## 注释

```lua
--最基础的注释


--[[******多行注释******]]
--[[
print("多行注释")
print("多行注释")
]]



--[[******开关注释******]]
--[[]]
print("多行注释")
print("多行注释")
--]]

--[[
print("多行注释")
print("多行注释")
--]]

--[[**单刀双掷开关注释***]]
--[=[版本A]=]
print("多行注释A")
--[[-[版本B]=]
print("多行注释B")
--]-]]


--[=[版本A]-=]
print("多行注释A")
--[[-[版本B]=]
print("多行注释B")
--]-]]
```
